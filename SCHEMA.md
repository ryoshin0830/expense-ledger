## 取引記録 (Transactions) — Sheet: `取引記録`

| # | Column | Type | Required | Description |
|---|---|---|---|---|
| 1 | id | UUID | ✅ | Unique identifier (auto-generated) |
| 2 | 日付 | DATE | ✅ | YYYY-MM-DD |
| 3 | 場所 | TEXT | ✅ | Store name / location |
| 4 | 品目 | TEXT | ✅ | Item description |
| 5 | 金額 | NUMBER | ✅ | Amount in original currency |
| 6 | 通貨 | SELECT | ✅ | JPY / USD / CNY / KRW / EUR / TWD / ... |
| 7 | JPY換算 | FORMULA | auto | `=E2*H2` (金額 × 為替レート) |
| 8 | 為替レート | NUMBER | auto | Exchange rate to JPY (1.0 for JPY) |
| 9 | 支払方法 | SELECT | ✅ | 現金 / クレジットカード / Wise / Alipay / WeChat Pay / PayPay / 銀行振込 / ... |
| 10 | カテゴリ | SELECT | ✅ | See `⚙️ 設定` sheet for master list |
| 11 | 事業区分 | SELECT | ✅ | 私用 / 個人事業主 / 法人(合同会社) / 正社員(会社名) |
| 12 | 経費区分 | SELECT | ✅ | 事業 / 私用 / 按分 |
| 13 | 精算状態 | SELECT | ✅ | 不要 / 申請前 / 申請中 / 精算済 |
| 14 | プロジェクト | TEXT | optional | Project/engagement name |
| 15 | 備考 | TEXT | optional | Notes |
| 16 | 作成日時 | TIMESTAMP | auto | ISO 8601 timestamp |

## Currency & Exchange Rates

- JPY is the base currency (rate = 1.0)
- Exchange rates stored in `為替レート` column
- `JPY換算` = `金額 × 為替レート` (formula)
- Rates fetched via `scripts/exchange_rate.py`
- Historical rates NOT tracked; current rate used at time of entry

## Data Integrity Rules

1. **id** is always UUID v4, auto-generated by `scripts/record.py`
2. **日付** must be valid YYYY-MM-DD
3. **通貨** must match a value in `⚙️ 設定!A:A` (通貨 column)
4. **事業区分** must be one of the 4 defined values
5. **経費区分** must be one of: 事業 / 私用 / 按分
6. **精算状態** must be one of: 不要 / 申請前 / 申請中 / 精算済
7. **為替レート** is auto-filled by `record.py` for non-JPY currencies
8. **JPY換算** is a sheet formula, not written by script
9. Duplicate detection: same date + place + amount + item within 60 seconds → warning
